# AI House

A browser-based top-down pixel art game where three LLM-powered characters — a husband, a wife, and a mischievous poltergeist — play out a 10-day drama. The poltergeist's goal is to sow discord between the couple. The player watches the story unfold.

Built with Phaser 3 + Vite. Each character is powered by a separately configurable LLM (any OpenAI-compatible endpoint).
